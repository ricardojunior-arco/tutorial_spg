Como incluir troca de chapa em um material de Professor

Made by Ricardo Junior with Scribe

Learn how to update the physical specifications and component settings for teacher materials within the platform. This guide walks you through navigating the component menu, adjusting paper quality and page counts, and saving your final configuration updates.



Workflow Initiation

1. Navigate to https://spg.stage.arcocv.co/components/2e63f990-aeae-4806-a279-f94863a4e7b6

2. Select 'Especificação Física' tab

3. Click 'Editar Especificação'

Miolo Configuration

4. Select 'Miolo Reaproveitado'

5. Enter the student material SKU 'TSTBLGG527'

6. Set the Magenta (M) plate color value to 100

7. Click "Próxima Etapa"

Component Detailing

8. Navigate to '2. Encarte 1' details

9. Set the color selection to '1x1'

10. Set the paper type to 'Offset 56g (56g)'

11. Navigate to '3. Miolo 1' details

12. Navigate to '4. Encarte 2' details

13. Set the component name to 'Manual do professor'

14. Set the pagination to 56

15. Set the paper type to 'Offset 56g (56g)'

16. Set the color selection to '1x1'

17. Click "Próxima Etapa"

Final Review and Save

18. Proceed to Stage 4 (Notas)

19. Proceed to Stage 5 (Resumo)

20. Click "Salvar Especificação"
