Como criar e compor um kit

Made by Ricardo Junior with Scribe

Learn the step-by-step process for setting up a composite BOM kit within the system. This guide covers defining kit specifications, adding specific items, and selecting the appropriate packaging for final assembly.



Kit Creation Setup

1. Navigate to https://spg.stage.arcocv.co/bom-kits

2. Click "Criar"

3. Select "Kit composto com múltiplos itens"

Kit Details Configuration

4. Type "TSTPESG5" in Kit Code field

5. Type "PES G5 KIT TESTE" in Description field

6. Select "PES - PES" from Marca dropdown

7. Select "27 - Coleção 27" from Coleção dropdown

8. Select "INF 5 ANOS" from Série dropdown

9. Select "Vol. 1/L1" from Volume dropdown

10. Select "Aluno" from Usuário dropdown

11. Select "Bimestral" from Frequência dropdown

12. Select "V1" from Envio dropdown

13. Click "Criar Kit"

Kit Composition

14. Click "Construir Kit"

15. Search for "biology" and select "Biology Grade 5 - Lab Manual"

16. Click "Adicionar Item" to add selection to kit

17. Search for "grade 5" and select "PES KIDS TIME A LVL A SAFARI 1 PR PLUS CUP PF"

18. Click "Adicionar Item" to add selection to kit

Packaging and Handling

19. Type "Colecionar em plástico shrink e em caixa parda" in handling instructions field

20. Select box "30000108 • CAIXA ARCO EMB PAPELAO TF 430x335x225"

21. Click "Confirmar caixa"

22. Click "Salvar Alterações" to finalize kit

23. Select "Instruções de manuseio" tab
