Como criar um novo produto no SPG

Made by Ricardo Junior with Scribe

This guide provides a step-by-step process for registering a new educational product within the SPG platform. Learn how to accurately configure product details, including grade levels, subject matter, and volume specifications to ensure a successful entry.



Navigation and Initialization

1. Navigate to https://spg.prod.arcocv.co/home

2. Click "Produtos" in the sidebar menu.

3. Click "+ Criar" to start a new product registration.

4. Select "Em branco" to start a new product registration.

Basic Product Information

5. Enter the product description and select as séries do dropdown.

6. Select "Gráfico" from the "Categoria" dropdown. Caso o produto seja não-gráfico, marque a opção.

7. Enter the product SKU in the "SKU Atual" field.

8. Click "Próximo" to proceed to the Classification step.

Product Classification

9. Select "Físico" as the Macrotype.

10. Select o tipo de material as the Type.

11. Select the Brand.

12. Select "Aluno" from the "Usuário" dropdown. para Productos com mais de um usuário é só selecionar todas as opções

13. Select the Subject (Matéria).

14. Select the Volume.

15. Select Envio do produto.

16. Select the Collection (Coleção).

17. Select the Frequency (Frequência).

Product Finalization

18. Click "Salvar Produto" to finalize the product creation.
