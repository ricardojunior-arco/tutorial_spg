Criando e especificando um material inclusivo

Made by Ricardo Junior with Scribe

Learn how to efficiently clone an existing product and configure it for an enlarged format. This guide walks you through setting specific physical specifications, material types, and finishing options to ensure your product meets production requirements.



Product selection

1. Navigate to https://spg.stage.arcocv.co/home

2. Click "Produtos"

3. Click "Criar"

4. Click "A partir de SKU" to clone an existing product.

5. Paste the reference SKU into the search field.

6. Select the reference product from the search results.

7. Click "Criar e abrir formulário"

Basic information

8. Update the product description by appending "- AMPLIADO 12" if needed.

9. Enter the complete new SKU in the "SKU Atual" field.

10. Click "Próximo"

Classification and usage

11. Toggle "Material Inclusivo" to "SIM".

12. Select "Ampliado" as the inclusive material type.

13. Enter "Em A3" in the observations field.

14. Click "Salvar Produto"

Physical specifications

15. Navigate to the "Especificação Física" tab.

16. Click "Adicionar Especificação"

17. Set the item dimensions to 297 mm (Largura) x 420 mm (Altura).

18. Check the box to include the "Capa" component.

19. Set "Qtd. blocos" to "1" for the Miolo component.

20. Click "Próxima Etapa"

Component configuration

21. Select "Couché fosco 170g (170g)" as the paper type for the Capa.

22. Set the "Tipo da Capa" to "Simples".

23. Select "4X4" as the "Cor de Impressão" for the Capa.

24. Click "2.Miolo 1"

25. Set the pagination for Miolo 1 to "120".

26. Select "Offset 63g (63g)" as the paper type for Miolo 1.

27. Select "4x1" as the "Cor" for Miolo 1.

28. Enter "Em papel azul" in the observations field for Miolo 1.

29. Click "Próxima Etapa"

Binding and finalization

30. Select "Espiral Plástico" as the finishing type.

31. Set "Posição do Acabamento" to "Lateral Esquerda".

32. Select "⌀ 9mm (lombada até 3,5mm)" as the bitola.

33. Select "Preto" as the spiral color.

34. Click "Próxima Etapa"

35. Click "Próxima Etapa"

36. Review the configuration and click "Salvar Especificação".
