Como atribuir um perfil a um usuário

Made by Ricardo Junior with Scribe

Learn how to navigate the user management interface to adjust permission settings for specific accounts. This guide provides a quick walkthrough to ensure your team has the correct portfolio viewing access.



1. Navigate to 'Início'.

2. Select 'Usuários' from the sidebar menu.

3. Click the Edit icon for the user you wish to modify.

4. Select 'Portfólio' from the Perfil dropdown menu.

5. Click 'Update' to save the changes.
