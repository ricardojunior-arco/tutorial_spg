Como atribuir um perfil a um usuário

Criado por Ricardo Junior com Scribe

Aprenda a navegar pela interface de gestão de usuários para ajustar as permissões de acesso de contas específicas. Este guia apresenta o passo a passo para garantir que sua equipe tenha o nível de acesso correto na plataforma.

Atribuição de Perfil

1

Acesse "Início".

2

Selecione "Usuários" no menu lateral.

3

Clique no ícone de edição do usuário desejado.

4

Selecione o perfil desejado no campo "Perfil".

5

Clique em "Atualizar" para salvar as alterações.
