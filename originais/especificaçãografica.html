Como construir uma especificação gráfica

Follow this guide to accurately input physical dimensions, paper types, and binding requirements for your product components. These steps will help you properly define technical specifications and finalize product configurations within the system.



Initial Setup

1. Navigate to https://spg.stage.arcocv.co/components/f5b183ac-2ea5-486f-9094-32403dc8cb53

2. Click "Especificação Física"

3. Click "Adicionar Especificação"

General Dimensions and Components

4. Enter dimensions: 205 for Width and 275 for Height

5. Select the 'Capa' checkbox to include it in the component list

6. Set the block quantity for the Miolo component

7. Click 'Próxima Etapa' to proceed to the details configuration

Component Detailed Configuration

8. Switch to 'Encarte 1' to configure its specific details

9. Select 'Cartão 250g (250g)' as the paper type for the cover

10. Set the cover type to 'Simples'

11. Apply 'Bopp Fosco' as the coating finish

12. Select 'Frente' for the finish application

13. Select '4X4' for the printing color

14. Click 'Próximo Componente' to advance through the selection

15. Enter '32' for the Encarte 1 pagination

16. Select 'Offset 70g (70g)' as the paper type for Encarte 1

17. Select '4X4' for the Encarte 1 printing color

18. Switch to 'Miolo 1' to configure its specific details

19. Enter '120' for the Miolo 1 pagination

20. Select 'Offset 60g (60g)' as the paper type for Miolo 1

21. Select '4X4' for the Miolo 1 printing color

22. Click 'Próxima Etapa' to move to the binding configuration

Binding and Packaging

23. Select 'Lombada quadrada com PUR' as the binding finish type

24. Select 'Lateral Esquerda' for the finish position

25. Select the recommended packaging box from the dropdown

26. Click 'Próxima Etapa' to proceed to the assembly notes

Finalization and Saving

27. Click 'Próxima Etapa' to review the summary

28. Click 'Salvar Especificação' to finalize the workflow
