Como criar um avulso

Made by Ricardo Junior with Scribe

Learn how to efficiently create and manage lab manual kits within your inventory system. This guide provides step-by-step instructions for adding specific products, configuring handling requirements, and saving changes for your educational materials.



Kit Creation via SKU

1. Navigate to Kits in the sidebar

2. Click + Criar

3. Select the Avulso kit option

4. Search for the product SKU and select the item from the results

5. Go to the Instruções de manuseio tab

Kit Creation via Products List

6. Navigate to the Produtos section

7. Select a product from the list

8. Click Criar avulso desse produto

9. Click Voltar para Kits

Kit Configuration

10. Select the kit from the list

11. Navigate to Instruções de manuseio, click Gerar instrução, and save changes

12. Click Prévio in the progress funnel
